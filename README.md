# 🚀 Mon Portfolio Personnel

Bienvenue sur le dépôt GitHub de mon portfolio en ligne. Ce projet présente mes compétences, mes réalisations et mon parcours en tant que développeur.

🔗 **[Voir le site en direct](https://impossibol04.github.io/Portfolio/)**

---

## 🧐 À propos
Ce portfolio a été conçu pour être à la fois simple, moderne et interactif. Il me sert de carte de visite numérique pour présenter mes projets à des clients ou recruteurs potentiels.

## 🛠️ Technologies utilisées
Pour réaliser ce projet, j'ai utilisé les technologies suivantes :

* **HTML5** : Structure sémantique du site.
* **CSS3** : Design responsive et animations (inclut `style.css`).
* **JavaScript (ES6+)** : Interactivité et dynamisme (inclut `script.js`).
* **GitHub Pages** : Hébergement gratuit et déploiement continu.

## ✨ Fonctionnalités
- [x] **Design Responsive** : Adapté aux mobiles, tablettes et ordinateurs.
- [x] **Interactivité** : Menu dynamique et effets au survol.
- [x] **Optimisation** : Chargement rapide et code propre.
